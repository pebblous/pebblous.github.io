## 핵심 정의

- **AADS (Agentic AI Data Scientist):** 데이터 품질 진단에서 개선까지 전 과정을 자율적으로 수행하는 에이전트형 AI 데이터 과학자. 과기정통부 '글로벌 빅테크 육성사업'으로 (주)페블러스(주관)와 KISTI(공동연구)가 2025년부터 3년간 총사업비 61억원으로 수행. [P1][P3][H1]
- **Data Greenhouse:** AADS의 차세대 진화 형태. "관측(Observe)–판단(Orchestrate)–행동(Action)–증명(Govern)"의 4단계 자율 순환 루프를 통해 데이터가 스스로 진단하고 치료하여 성장하는 무인화 데이터 운영 체계. 기존 데이터 플랫폼(Snowflake, Databricks 등) 위에 얹혀져 데이터 운영의 책임을 지는 '책임 레이어(Responsibility Layer)'. [P4][M3]
- **PebbloSim (Data Synthesis):** 페블러스의 합성 데이터 생성 기술. Data Diet(중복 제거), Data Bulk-up(합성 생성), Data Replica(통계적 섭동 기반 비식별화)를 포함. [P1][P3]
- **데이터 클리닉 (Data Clinic):** 페블러스의 데이터 품질 진단 서비스. Neural(임베딩) + Symbolic(온톨로지) 이중 분석으로 데이터의 과밀·공백·이상치를 탐지. AADS로 소프트웨어화하여 완전 자동화 추진 중. [P1][P6]
- **뉴로-심볼릭 AI (Neuro-Symbolic AI):** 신경망의 통계적 학습 능력과 심볼릭 AI의 논리적 추론 능력을 결합한 아키텍처. AADS의 핵심 기술 기반으로, 데이터 임베딩(Neural)과 온톨로지(Symbolic)를 통합하여 설명 가능하고 신뢰할 수 있는 진단-처방을 수행. [P5]

## 소스별 Key Facts

### [P1] AADS 기술 기반 데이터 클리닉 2.0 및 AI-Ready 데이터 플랫폼 비전.pdf

#### 정의 / 개념
- AADS는 "자율형 AI 데이터 과학자"로, 데이터 품질 진단에서 개선까지 전 프로세스를 자동으로 수행
- 데이터 클리닉 2.0: 기존 데이터 클리닉의 진화형으로, Neural(임베딩 기반 분석) + Symbolic(온톨로지 기반 해석)의 뉴로-심볼릭 접근
- AI-Ready 데이터 플랫폼: 데이터를 AI 학습에 최적화된 상태로 준비하는 종합 플랫폼

#### 핵심 수치
- 데이터 과학자 업무 시간의 80%가 데이터 준비 작업에 소요
- AADS 핵심 도구: 데이터 진단, 데이터 다이어트, 데이터 벌크업, 데이터 레플리카

#### 인용 가능 문장
> [P1-Q1] "기업들은 '데이터 부채'에 허덕이고 있으며, 데이터 과학자들은 업무 시간의 80%를 수동적인 데이터 준비 작업에 소요하고 있습니다."
> [P1-Q2] "AADS는 '진단(Diagnosis) → 처방(Prescription) → 실행(Action) → 검증(Verification)'의 자율 순환 루프를 통해 데이터 품질 문제를 해결합니다."

#### 아키텍처 요소
- 뉴로-심볼릭 진단 엔진 (Neural 임베딩 + Symbolic 온톨로지)
- Data Diet (중복·저품질 데이터 제거)
- Data Bulk-up (합성 데이터 생성으로 클래스 불균형 해결)
- Data Replica (통계적 섭동 기반 비식별화)
- PebbloScope (데이터 가시화)

#### 한계점
- 문서에서 직접 언급한 한계: 아직 완전 자동화 단계 도달 전 (PoC 단계)

### [P3] 251212_AADS 1차년도 발표자료_v1.11 main.pdf

#### 정의 / 개념
- AADS 1차년도(2025) 사업 성과 발표
- 데이터 품질 진단 → 처방 → 실행의 에이전트 루프 PoC
- 로봇·제조·규제·사회안전 4개 도메인 LLM 파인튜닝용 QA 데이터셋 구축

#### 핵심 수치
- 4개 도메인 (로봇, 제조, 규제/거버넌스, 사회안전) QA 데이터셋 구축
- 로봇 분야: 13개 데이터셋 그룹, 4가지 질의 유형
- 제조 분야: 14개 논리적 데이터 그룹, 28개 QA 쌍
- 주관기관: (주)페블러스 / 공동연구: KISTI
- 로드맵: 2025 Q4 핵심 에이전트 루프 PoC → 2026 Q2 파일럿 → 2026 Q4 베타 출시

#### 인용 가능 문장
> [P3-Q1] "AADS 과제의 핵심 자산으로서, 로봇 분야의 복잡한 맥락과 기술적 뉘앙스를 깊이 있게 이해하는 전문 LLM 개발의 초석이 될 것입니다."
> [P3-Q2] "에이전트(Agentic) AI 데이터 과학자를 구현하는 데 결정적으로 기여할 것입니다."

#### 아키텍처 요소
- AADS 에이전트 루프: 플래너(Planner) → 도구(Tools) → 실행(Execution)
- 전문 도구: 데이터 진단, 데이터 다이어트, 데이터 벌크업
- LLM 파인튜닝 파이프라인: 도메인 문서 → QA 데이터셋 → 커스텀 LLM
- CLI 기반 워크플로우 시뮬레이션

#### 한계점
- 1차년도는 PoC 단계, 대규모 QA 데이터셋 확장은 향후 과제
- 실제 산업 환경 적용 검증은 2차년도 목표

### [P4] 2025 데이터 품질관리 시장 분석 및 페블러스 AADS 차세대 전략.pdf

#### 정의 / 개념
- 가트너(Gartner) AI와의 대화를 통한 시장 분석
- 데이터 품질관리 시장의 4대 통합 패턴 분석
- Data Greenhouse: 자율 순환 루프 기반 데이터 운영 체계

#### 핵심 수치
- 2025년 데이터 관리 소프트웨어 시장 규모: $69.2B (약 69.2조원)
- 가트너 4대 통합 패턴: ① 결합 서비스 ② TDM+합성 대체 ③ 도메인 특화 ④ 전문가 주도형
- Reasoning Router: 추론 비용 70% 절감 목표

#### 인용 가능 문장
> [P4-Q1] "가트너는 현재 시장의 스타트업들이 해결해야 할 핵심 과제로 '진단과 개선의 긴밀한 통합', '완전 자동화', '신뢰성 확보'를 꼽았습니다."
> [P4-Q2] "2025년 AI 시장은 기술적 가능성을 탐색하는 단계를 지나, 실질적인 산업 적용과 생존을 위한 '대전쟁(The Great Competition Wars)' 국면으로 진입했습니다."
> [P4-Q3] "AI 개발의 병목이 '모델'이 아닌 '데이터 품질'임이 명확해짐에 따라, 데이터 품질 관리 및 거버넌스 도구가 AI 생태계의 핵심적인 '곡괭이와 삽(Picks and Shovels)'으로 평가받고 있습니다."
> [P4-Q4] "가트너는 페블러스를 '결합 서비스 모델(Paired Services Model)'의 대표 사례로 명시하였습니다."

#### 아키텍처 요소
- Data Greenhouse 4단계 자율 순환 루프:
  1. Observe (관측): Platform Adapter Layer + Observation Layer (Neural+Symbolic)
  2. Orchestrate (판단): AADS + Human-in-the-Loop 승인 게이트
  3. Action (행동): Data Diet / Data Bulk-up (GenQA, Gen-VLM) / Data Replica / RAG Optimization
  4. Govern (증명): ISO/IEC 5259, ISO 42001 기반 증적 자동화
- Platform Adapter Layer: Snowflake/Databricks/DataLake 연결
- Reasoning Router: sLLM ↔ 거대 모델 자동 분배

#### 한계점
- 시장 분석 기반 전략 문서, 기술 구현 상세는 제한적

### [P5] 뉴로-심볼릭 AI: 엔터프라이즈 인텔리전스를 위한 코그니티브 데이터 아키텍처 v1.1.pdf

#### 정의 / 개념
- 뉴로-심볼릭 AI: 신경망(Neural)의 통계적 학습과 심볼릭 AI의 논리적 추론을 결합
- 코그니티브 데이터 아키텍처: 지식 그래프 + 임베딩 공간의 통합 운영
- 헨리 카우츠(Henry Kautz)의 AI 분류 체계 참조

#### 핵심 수치
- AI 분류 6단계 (카우츠 분류): 심볼릭 → 통계적 → 뉴럴 → 커넥셔니스트 → 뉴로-심볼릭 → 에이전트
- AADS에서 뉴로-심볼릭 접근의 적용: 온톨로지(도메인 지식) + 임베딩(데이터 패턴)

#### 인용 가능 문장
> [P5-Q1] "뉴로-심볼릭 AI는 신경망의 '빠른 사고(Fast Thinking)'와 심볼릭 AI의 '느린 사고(Slow Thinking)'를 결합하여, 설명 가능하고 신뢰할 수 있는 AI 시스템을 구현합니다."
> [P5-Q2] "데이터 품질 진단에 뉴로-심볼릭 접근을 적용하면, 단순한 통계적 이상 탐지를 넘어 '왜 이 데이터가 문제인지' 인과적으로 설명할 수 있습니다."

#### 아키텍처 요소
- Neural 레이어: 데이터 임베딩, 밀도 분석, 이상치 탐지
- Symbolic 레이어: 온톨로지, 지식 그래프, 규칙 기반 추론
- 통합 추론 엔진: Neural 관측 + Symbolic 해석 = 설명 가능한 진단
- 코그니티브 데이터 아키텍처: 지식 표현과 학습의 통합

#### 한계점
- 뉴로-심볼릭 통합의 실제 구현 복잡성 (두 패러다임의 이질성)
- 대규모 온톨로지 구축 및 유지의 비용

### [P6] AI 데이터 품질 표준과 페블러스 데이터클리닉.pdf

#### 정의 / 개념
- ISO/IEC 5259 시리즈: AI 데이터 품질 국제 표준
- 데이터클리닉과 ISO 5259 표준의 매핑
- 데이터 품질 측정 기준(Quality Measures): 정확성, 완전성, 일관성, 적시성 등

#### 핵심 수치
- ISO/IEC 5259-2: 데이터 품질 측정 기준(QM) 체계
- ISO 42001: AI 관리 시스템 표준
- 데이터클리닉의 진단 항목과 ISO 5259 품질 지표 간 1:1 매핑

#### 인용 가능 문장
> [P6-Q1] "ISO/IEC 5259는 AI 학습 데이터의 품질을 체계적으로 측정하고 관리하기 위한 국제 표준으로, 데이터의 정확성·완전성·일관성·적시성 등을 정량화합니다."
> [P6-Q2] "페블러스 데이터클리닉은 ISO/IEC 5259 표준의 품질 측정 기준을 내재화하여, 진단 결과가 국제 표준에 부합하는 품질 증적(Evidence)으로 활용될 수 있습니다."

#### 아키텍처 요소
- ISO/IEC 5259-1: 개요 및 용어 정의
- ISO/IEC 5259-2: 데이터 품질 측정 기준(QM)
- ISO/IEC 5259-3: 데이터 품질 관리 요구사항
- ISO/IEC 5259-4: 데이터 품질 프로세스 프레임워크
- 데이터클리닉 ↔ ISO 5259 매핑: 진단 항목별 표준 연계

#### 한계점
- 표준 자체가 아직 진화 중 (일부 파트 Draft 단계)
- 실제 인증 프로세스의 복잡성

### [P7] Physical AI 시대의 패권 경쟁: 데이터 중심 생존 전략과 페블러스의 역할.pdf

#### 정의 / 개념
- Physical AI: 디지털을 넘어 물리적 세계와 상호작용하는 AI (제조, 로봇, 국방 등)
- 디지털 트윈과 Physical AI의 융합
- 데이터 중심(Data-Centric) AI 접근의 중요성

#### 핵심 수치
- Physical AI 시장의 급성장 (제조, 로봇, 국방 분야)
- 고난이도 멀티모달 데이터 수요 폭증
- 소버린 AI(Sovereign AI): 데이터 안보와 기술 자립 트렌드

#### 인용 가능 문장
> [P7-Q1] "Physical AI 시대에는 디지털 챗봇을 넘어 제조, 로봇, 국방 등 물리적 세계와 상호작용하는 AI가 주류로 부상하며, 현실 세계의 복잡한 변수를 반영한 고난이도 멀티모달 데이터 수요가 폭증합니다."
> [P7-Q2] "데이터 품질 관리 소프트웨어는 AI 생태계의 '곡괭이와 삽(Picks and Shovels)'으로, Physical AI 시대의 핵심 인프라입니다."

#### 아키텍처 요소
- Physical AI 데이터 파이프라인: 센서 → 멀티모달 데이터 → VLM 학습
- AADS의 Physical AI 특화: Gen-VLM (시각적 상황 인지 합성 데이터)
- 소버린 배포: 온프레미스 패키지로 국방/공공 시장 대응

#### 한계점
- 저자가 인정: Physical AI 데이터의 복잡성으로 인한 품질 관리 난이도 상승

### [M2] AADS LLM 파인튜닝용 QA 데이터셋 구축: 제조 분야.md

#### 정의 / 개념
- 제조 도메인 LLM 파인튜닝을 위한 QA 데이터셋 구축
- 14개 논리적 데이터 그룹 (OHT/AGV 탄화 예지보전, 김치 품질검사, 3D 프린팅, 금속 3D 프린팅, 건설기계, 리튬이온전지, 가전기기 전력, CMF 이미지, 재료 물성, 배터리 불량, LNG 탱크, P&ID 심볼, 선박 도장, 용접 AI)

#### 핵심 수치
- 14개 데이터 그룹, 28개 QA 쌍
- 질의 유형 균등 배분: A 7회(25%), B 7회(25%), C 7회(25%), D 7회(25%)
- 주요 AI 모델: MultiTask EfficientNetV2, Swin Transformer, YOLOv5x-seg, Mask2Former, Mask DINO, LLama3.2-3B
- 주요 성능지표: F1-score, mAP, mIoU, PQ, Accuracy

#### 인용 가능 문장
> [M2-Q1] "제조 도메인 데이터 과학의 핵심 요소인 목적(A), 데이터 특징(B), 기술 적용(C), 품질 관리(D) 영역에 대해 균등하게 질문을 배분하여 LLM이 전 영역에 걸친 종합적인 지식을 학습하도록 설계되었습니다."

#### 주요 제조 데이터셋 수치
- 3D 프린팅 외형품질이미지: 165,780장 / 수축분석: 55,260장
- 금속 3D 프린팅 스파크 이미지: 51,267,476개
- 건설기계 활동인식 F1-score: 89.98%, 팬옵틱 PQ: 78.41%
- 배터리 CT 결함 검출 mIoU: 92.79%
- LNG 탱크 세그멘테이션 mAP: 95.72%
- 가전기기 NILM 상태분류 F1-score: 95.5%
