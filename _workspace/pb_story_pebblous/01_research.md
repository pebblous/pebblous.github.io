# 리서치: 나는 페블러스입니다 (Pebblous pb Story)

## 화자 프로필 (Speaker Profile)

화자: 페블러스(Pebblous) 자신
- 2021년 11월 대한민국 ETRI 출신 연구자들이 창업한 딥테크 스타트업
- CEO: 이주행 (POSTECH 컴퓨터공학 박사, ETRI 책임연구원 출신)
- COO: 이정원 (KAIST 뇌공학 박사, ETRI 선임연구원 출신)
- 슬로건: "Make Data Tangible" — 데이터를 만질 수 있는 형태로
- 직원 약 14명의 소규모 팀, 하지만 글로벌을 지향

## 탄생 드라마 (Birth Drama)

### 문제의식
로봇, 자율주행차, 스마트팩토리 등 피지컬 AI의 시대가 열리고 있지만, 이를 훈련시킬 데이터가 문제였다.
- 현실 데이터 수집: 비싸고, 느리고, 위험하고, 불완전
- 데이터 품질 진단: 눈에 보이지 않는 고차원 수치 — "이게 좋은 데이터인지 어떻게 알지?"
- ETRI에서 수년간 제조 AI, 자율주행 데이터를 다루면서 쌓인 좌절

### 창업 계기
이주행 대표는 ETRI에서 공장 AI 데이터를 다루면서 "데이터를 눈으로 볼 수 있다면, 만질 수 있다면"이라는 아이디어를 갖게 됨.
기하학적 다양체(Geometric Manifold)로 고차원 데이터를 시각화하는 "Data Imaging" 기술.
→ 2021년 11월, ETRI 동료들과 함께 페블러스 창업.

## 3대 핵심 제품 (Core Products Triangle)

### 1. DataClinic (데이터 클리닉) — AI 데이터 품질 진단 SaaS
- 고차원 데이터를 기하학적 다양체로 시각화해 결함 진단
- 10만장/시간 진단 처리 능력
- Level 1 (기초 통계), Level 2 (DataLens 뉴럴넷 매니폴드), Level 3 (커스텀 도메인)
- 검증 고객: 현대차, LG전자, LGU+, 한화비전, 삼성E&A, 육해공군
- "데이터의 건강검진" — 눈에 보이지 않던 문제를 시각화

### 2. Data Greenhouse (데이터 그린하우스) — 피지컬 AI를 위한 데이터 OS
- 데이터를 "수집(hunting)"하는 게 아니라 "경작(cultivation)"하는 패러다임 전환
- 자율 데이터 운영 체계: 자가 모니터링 → 평가 → 조정 루프
- 씨앗(seed data)에서 시작해 합성·증강으로 성장하는 "온실" 모델
- 데이터 재배 파이프라인: 원천 → 정제 → 증강 → 평가 → 배포

### 3. PebbloSim — 합성 데이터 생성기 (Synthetic Data Generator)
- 피지컬 AI를 위한 뉴로-심볼릭(Neuro-Symbolic) 합성 데이터
- 물리적 법칙 기반 생성 → 물리적 환각(physical hallucination) 제거
- Sim-to-Real 갭 해소: 시뮬레이션 데이터가 현실에서 그대로 통하는 이유
- 설명 가능한 인과관계(Explainable Causality) 보장
- 엣지 케이스 자동 생성 (실제로 일어나기 전에 준비)

## 핵심 데이터 포인트 (Key Facts)

### 시장 규모
- 글로벌 합성 데이터 시장: CAGR 31~46%
  - 2025년: 50~90억 달러 (약 7~13조원)
  - 2030년: 250~340억 달러 (약 36~49조원)
- 피지컬 AI 시장: 2025년 510~540억 달러 → 2033~35년 5,000~8,400억 달러

### 지식재산권
- 한국 특허: 36개+ 출원, 5개+ 등록
- 미국 특허: 4개+ 출원 (2025년 11월 US 12,481,720 등록)
- PCT 국제 출원
- 기하학적 다양체 기반 데이터 진단 핵심 IP 보유

### 정부 지원 및 트랙션
- 61억원 규모 정부 과제 주관기관 선정
- 10조원 정부 AI 예산 생태계 참여
- Hyundai Motor, LG Electronics, LGU+, Hanwha Vision, Samsung E&A, 국군 등 엔터프라이즈 고객

### 경쟁 우위
- NVIDIA Isaac, Applied Intuition 등 글로벌 플레이어 대비:
  - 유일한 통합 플레이어: 데이터 OS + 품질 진단 + 시뮬레이션 기반 합성 데이터
  - 한국 제조업 특화 DNA (자동차, 반도체, 방산)
  - 설명 가능한 AI (XAI) 기반 — 블랙박스가 아닌 이유를 알 수 있는 진단

### 자부심

- 세계 최초 기하학적 다양체 기반 데이터 진단 특허 보유
- 현대차, 삼성이 우리 손을 빌린다는 사실

- 14명의 팀이 NVIDIA와 경쟁한다는 현실
- "데이터 사냥꾼"의 시대에 "데이터 농부"가 되겠다는 선언

- 작은 팀이지만 특허는 글로벌, 고객은 대기업
