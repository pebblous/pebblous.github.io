## 핵심 결론

**GSC가 보여주는 CTR 0.75%는 봇 노출에 의해 왜곡된 수치다. 진짜 사람 CTR은 약 5%로, 업계 평균(2~3%) 이상이다.**

비평글의 "CTR이 업계 평균보다 낮다"는 진단은 **봇 노출로 분모가 부풀려진 결과**였다. 콘텐츠 전략 진단(브랜드 검색·포트폴리오 집중)은 여전히 유효하지만, CTR 자체를 문제로 다룰 필요는 없다.

1. **GSC: "피지컬ai란 무엇인가" → Position 1.00 / 288 impressions / 0 clicks / 0% CTR**
2. **사용자 시크릿 모드 검색 결과: 따옴표 묶음 시 9위, 일반 검색 시 안 보임. AI Overview에도 없음.**

GSC가 1위로 기록했는데 실제 SERP에서 우리 페이지가 없다 → "Position 1"은 사람 검색 결과가 아니라 **봇이 만들어낸 노출**일 가능성.

## 한계와 주의사항

1. **추정의 정확도**: 모바일 CTR 4.91%를 "진짜 사람 CTR"로 가정한 단일 변수 기반. 데스크탑 사람은 검색 의도가 더 깊어 CTR이 더 낮을 수 있음 (현실은 3~5% 사이일 가능성).

2. **상위 1000 쿼리 한계**: GSC Queries.csv는 상위 1000개만 export. 전체 노출 218K 중 20K만 쿼리별로 분석 가능. 나머지 198K는 long-tail로 묶여 정체 추정만 가능.

3. **봇 패턴 휴리스틱의 정확도**: hash ID, arxiv ID, 긴 자연어 문장 등 휴리스틱은 false positive 가능 (사람이 정말 그런 검색을 하는 경우도 있음). 다만 임팩트 측면에서는 정확히 정수 Position + 클릭 0인 케이스가 가장 강력한 신호.

4. **GSC 자체의 봇 필터**: GSC는 일부 봇을 사전 필터링하지만 모든 봇을 잡지는 못함. 우리가 보는 218K는 이미 일부 필터링된 후의 수치.

## 새로운 KPI 가이드라인

향후 GSC 데이터를 인용할 때 다음 원칙을 따른다:

### 사용 권장 KPI
- **모바일+태블릿 CTR**: 가장 깨끗한 사람 신호
- **디바이스별 분리된 CTR**: 데스크탑은 봇 영향 명시
- **정확 매칭 1-3위 쿼리의 CTR**: 진짜 검색 의도 매칭 측정
- **클릭수 절대값**: 봇은 클릭 안 함, 클릭은 거의 다 사람

### 사용 주의 KPI
- **전체 CTR**: 봇 노출로 분모 부풀려짐, 단독 인용 금지
- **노출수**: 진짜 도달이 아님, "도달 = 노출 × 0.15" 정도로 추정
- **평균 Position**: 정수 Position 쿼리들이 평균을 왜곡

### 비교 시점
- 봇 트래픽은 콘텐츠 양과 비례 증가 (콘텐츠 늘면 학습 크롤러 증가)
- **MoM 비교 시 사람 KPI(모바일 CTR, 클릭 절대값)만 사용**
