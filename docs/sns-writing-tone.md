# Pebblous SNS 홍보 어조 가이드

SNS 콘텐츠 작성 시 사용하는 어조 및 스타일 가이드.

---

## 어조 이름: 따뜻한 전문가 어조 (Warm Expert Tone)

**별칭**: 스토리텔링 비즈니스 어조

---

## 핵심 특징

| 구분 | 설명 |
|------|------|
| **문체** | 합쇼체 기반 + "~이죠" 부드러운 전환 |
| **이모지** | 최소 사용 (핵심 포인트에만 1-2개) |
| **구조** | 역설 → 문제 제기 → 해결책 → 메타포 마무리 |
| **메타포** | 농업/온실 은유 일관 유지 (경작, 재배, 수확 등) |
| **톤** | 권위있되 딱딱하지 않음, 신뢰감 + 친근함 |

---

## 대표 예시

```
🌱 데이터를 수집하지 않습니다. 재배합니다.

피지컬 AI 시대, 가장 큰 역설이 있습니다.

AI가 '안전'을 배우려면 '위험한 순간'의 데이터가 필요한데,
현실에서 그런 사고를 일부러 낼 수는 없다는 것이죠.

페블러스는 이러한 '데이터 기근'을 아래와 같이 해결합니다.

🔸 물리 법칙을 완벽히 복제한 디지털 트윈: PebbloSim
🔸 물리적 환각 없는 뉴로-심볼릭 합성 데이터
🔸 자율 에이전트가 관리하는 데이터 품질

마치 숙련된 농부가 스마트 온실에서 초고품질의 토마토를 재배하듯,
페블러스는 《데이터 그린하우스》 에서 AI 학습 데이터를 경작합니다.
```

## 작성 원칙

### 1. 문장 구조

- **짧은 문장** 선호 (한 문장에 하나의 아이디어)
- **줄바꿈** 적극 활용 (시각적 호흡)
- **역설이나 질문**으로 시작하여 호기심 유발

### 2. 어미 패턴

| 용도 | 어미 예시 |
|------|----------|
| 선언/단정 | ~입니다, ~합니다 |
| 부드러운 전환 | ~이죠, ~거죠 |
| 강조 | ~것입니다, ~할 수 있습니다 |

### 3. 이모지 사용 규칙

- **허용**: 헤드라인에 1개 (🌱, 🚀, 💡 등)
- **허용**: 리스트 불릿 대체 (🔸, ▸)
- **금지**: 문장 중간 삽입
- **금지**: 과다 사용 (3개 이상 연속)

### 4. 메타포 일관성

페블러스의 핵심 메타포는 **"데이터 농업"**입니다.

| 개념 | 메타포 |
|------|--------|
| 합성 데이터 생성 | 재배, 경작 |
| 데이터 수집 (기존 방식) | 사냥, 채집 |
| Data Greenhouse | 스마트 온실 |
| Edge Case 부족 | 데이터 기근 |
| 고품질 데이터 | 수확물, 작물 |

## 플랫폼별 조정

### LinkedIn

- 전문성 강조
- 업계 맥락 언급 (규제, 트렌드)
- 해시태그 3-5개

### Twitter/X

- 핵심 메시지만 압축
- 링크는 마지막에
- 해시태그 1-2개

### Facebook

- 스토리텔링 확장 가능
- 질문형 마무리로 참여 유도
