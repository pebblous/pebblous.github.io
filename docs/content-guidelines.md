# Pebblous 콘텐츠 작성 가이드라인

블로그 포스트 및 기술 문서 작성 시 참조하는 핵심 원칙.

---

## 1. 텍스트-비주얼 순서 원칙 (Text-First, Visual-Second)

### 핵심 규칙

> **비주얼(차트, 카드, 다이어그램) 앞에는 반드시 설명 문단을 배치한다.**

### 왜 중요한가?

| 문제 | 해결 |
|------|------|
| 비주얼만 있으면 초보자는 해독 불가 | 텍스트로 맥락 제공 후 비주얼로 요약 |
| 정보가 기억에 남지 않음 | Dual Coding: 언어 + 시각 조합으로 기억 정착률 향상 |
| 도메인 지식 없이 접근 불가 | 설명 문단이 진입 장벽을 낮춤 |

### 올바른 패턴

```
❌ Bad: 비주얼만 덩그러니
┌─────────────────────┐
│   [차트/카드/표]     │
└─────────────────────┘

✅ Good: 설명 → 비주얼
┌─────────────────────┐
│ 이 차트는 X를 보여   │
│ 줍니다. Y가 핵심이며 │
│ Z에 주목하세요.      │
├─────────────────────┤
│   [차트/카드/표]     │
└─────────────────────┘
```

### 실제 적용 예시

**Before (문제)**:
```html
<h3>2.3 핵심 수치</h3>
<div class="grid grid-cols-3">
    <div class="stat-card">$69.2B</div>
    <div class="stat-card">4가지</div>
    <div class="stat-card">100%</div>
</div>
```

**After (개선)**:
```html
<h3>2.3 핵심 수치로 보는 시장</h3>
<p>가트너의 분석을 종합하면, 2025년 데이터 품질관리 시장의
핵심 포인트는 다음 세 가지로 요약됩니다. 먼저, $69.2B 규모의
데이터 관리 소프트웨어 시장은 AI 시대의 핵심 인프라로...</p>
<div class="grid grid-cols-3">
    <div class="stat-card">$69.2B</div>
    <div class="stat-card">4가지</div>
    <div class="stat-card">100%</div>
</div>
```

## 2. 소스 문서 활용 원칙

### 핵심 규칙

> **MD 소스 문서에 이미 고품질 문장이 있다면, 최대한 그대로 활용한다.**

### 작업 흐름

1. 소스 MD 문서를 먼저 읽고 핵심 문장 파악
2. HTML 변환 시 문장을 그대로 유지 (요약하지 않음)
3. 비주얼은 문장의 **보조 수단**으로 배치
4. 문장이 부족한 경우에만 새로 작성

### 주의사항

- 비주얼 중심으로 구성하면 "뒤죽박죽" 느낌
- MD 텍스트가 **내러티브의 백본(backbone)**
- 차트/카드는 핵심만 추출한 **요약 역할**

## 3. 섹션 구조 원칙

### 섹션 넘버링

```
1. 개요
2. 시장 동향
   2.1 세부 주제 A
   2.2 세부 주제 B
   2.3 핵심 수치
3. 다음 섹션
```

### 각 섹션의 필수 구성

1. **섹션 헤더** (아이콘 + 제목)
2. **도입 문단** (1-2개) - 이 섹션이 다루는 내용 소개
3. **본문 내용** (텍스트 + 비주얼 조합)
4. **마무리** (선택) - 다음 섹션으로의 연결

## 4. 비주얼 요소별 설명 가이드

### 차트 (PebblousChart)

```html
<p>이 차트는 [무엇]을 보여줍니다. [핵심 인사이트]가
두드러지며, 특히 [강조점]에 주목할 필요가 있습니다.</p>
<canvas id="myChart"></canvas>
```

### 카드 그리드 (stat-card, themeable-card)

```html
<p>[카드들이 요약하는 내용]을 세 가지로 정리하면 다음과 같습니다.
첫째, [A]. 둘째, [B]. 셋째, [C].</p>
<div class="grid grid-cols-3">
    <div class="stat-card">A</div>
    <div class="stat-card">B</div>
    <div class="stat-card">C</div>
</div>
```

### 탭 (PebblousTabs)

```html
<p>[탭들이 분류하는 카테고리] 중 [핵심 카테고리]가
가장 중요합니다. 아래 탭에서 각각의 상세 내용을 확인하세요.</p>
<div data-tabs>...</div>
```

### 레이어 스택 (layer-stack)

```html
<p>시스템은 N개의 레이어로 구성됩니다.
[첫 번째 레이어]는 [역할], [두 번째 레이어]는 [역할]...</p>
<p>각 레이어를 클릭하여 상세 내용을 확인하세요.</p>
<div class="layer-stack">...</div>
```
