## LinkedIn

GitHub 트렌딩에 같은 프로젝트가 한 달 만에 다시 1위로 올라왔습니다. Hermes Agent. +2,065 stars/day. 누적 14만 7천.

신규 프로젝트의 일시적 하이프라면 4월 정점 이후 식었어야 했죠. 그런데 한 달 사용 뒤에도 트래픽이 사라지지 않았습니다. 무엇이 다른가.

"함께 성장하는 에이전트(grows with you)"는 마케팅 카피가 아니라 아키텍처 선언입니다. Task Execution → Outcome Evaluation → Skill Abstraction → Skill Refinement으로 이어지는 4단계 자가 학습 루프가 매 태스크마다 작동합니다. 스킬은 라이브러리에 누적되고, 사용자 선호는 Honcho의 변증법적 프로파일링으로 진화하며, 세션이 끝나도 학습은 살아남습니다. LangChain의 무상태도, CrewAI의 제한적 메모리도, AutoGPT의 자기 프롬프팅도 한꺼번에 넘어선 첫 오픈소스입니다.

페블러스가 발행한 이번 리포트는 한 달 사이 NousResearch가 어떻게 응답했는지를 사실 기반으로 짚었습니다. v0.12.0의 Autonomous Curator는 죽은 스킬을 7일 주기로 자동 정리하면서 Error Fossilization 위험을 직접 겨냥했고, v0.13.0의 Hallucination Gate는 자기 평가 실패를 부분적으로 보완했습니다. 5개 릴리스, 4,000+ 커밋의 무게입니다. 다만 Distribution Shift는 미해결로 남았고, 모든 새 메커니즘은 여전히 에이전트 내부에서 작동합니다.

다음 AI 전쟁의 승부처는 모델이 아니라 자율형 데이터 운영체제입니다. 페블러스의 Data Greenhouse는 진단(DataClinic) → 생성(PebbloSim) → 관리(Data Greenhouse) → 증적의 통합 루프로, 자가 진화 에이전트의 외부 검증자 역할을 합니다.

## LinkedIn (EN)

The same project hit #1 on GitHub trending again, one month later. Hermes Agent. +2,065 stars/day. 147,000 cumulative.

If this were ordinary new-project hype, the wave should have receded after April's peak. It didn't. Why?

"Grows with you" turns out to be an architectural claim, not a tagline. A four-stage closed learning loop — Task Execution, Outcome Evaluation, Skill Abstraction, Skill Refinement — runs on every single task. Skills accumulate in a searchable library. User preferences evolve through Honcho's dialectical profiling. Learning persists across sessions. This is the first open-source agent to surpass LangChain's statelessness, CrewAI's limited memory, and AutoGPT's self-prompting in a single architectural shift.

Pebblous's new report examines, fact by fact, how NousResearch responded over the past month. v0.12.0's Autonomous Curator directly targets Error Fossilization with a 7-day automated prune cycle. v0.13.0's Hallucination Gate partially addresses self-evaluation failure. Five releases, over 4,000 commits in one month. Yet Distribution Shift remains unaddressed, and every new safeguard still operates inside the agent — never outside it.

The next AI battleground isn't the model. It's the Autonomous Data OS. Pebblous's Data Greenhouse closes that loop from the outside: diagnose (DataClinic), generate (PebbloSim), operate (Data Greenhouse), and document.
