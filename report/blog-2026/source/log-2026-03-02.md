**Before (보고서 기준)**:
```
Source MD → /new-post → /seo-check → articles.json → /publish → GitHub Actions
                                       ↑ /bilingual (옵션)
                                       ↑ /sns-write (옵션)
```

**After (현재)**:
```
Source MD → /new-post → /seo-check → articles.json → /publish → GitHub Actions
              ↑ /bilingual (대규모 배치)               ↑ /changelog (자동 기록)
              ↑ /text-reinforce (품질 강화)
              ↑ /new-hub (허브 페이지 자동 생성)         ← NEW
              ↑ /sns-write (SNS 홍보)
```

### 5.2 신규 파이프라인 패턴

#### 대규모 이중언어 배치 변환
```
1. 대상 선정 (category/project별)
2. /bilingual × N (배치 단위, 보통 4~5개)
3. articles.json 일괄 등록 (KO/EN 각각)
4. SEO redirect stub 생성 (구 .html → 신 ko/en/)
5. /commit (배치 단위 커밋 메시지)
6. /publish → CI 자동 (sitemap, rss, files-index)
```

#### 허브 페이지 생성
```
1. /new-hub (주제 키워드 지정)
2. PebblousHubCards.init() 설정
3. /seo-check (4-Layer 검증)
4. OG 이미지 생성
5. /commit + /publish
```

## 6. 주요 이슈 및 해결

| 이슈 | 원인 | 해결 |
|------|------|------|
| redirect stub 경로 오류 | 상대경로 `./ko/` → trailing slash 없으면 잘못 해석 | 절대경로 `/project/IR/ko/` 로 전환 |
| IR 허브 카드 1개만 표시 | KO PitchBook 기사가 articles.json에 미등록 | 누락 항목 추가 |
| AADS 설명 완전 오류 | "AI-ready Autonomous Data System" (틀림) | "Agentic AI Data Scientist" (정정) |
| FAQ JSON-LD 중복 | `<head>` 수동 삽입 + config.faqs 자동 삽입 → 이중 | 수동 삽입 제거, config.faqs만 사용 |
| articles.json wrapper 깨짐 | 편집 중 배열로 변환됨 | `{ categories, articles }` 래퍼 복구 |
| 인덱스 카테고리 "Loading..." | 비동기 렌더링 타이밍 이슈 | 렌더링 순서 수정 |
