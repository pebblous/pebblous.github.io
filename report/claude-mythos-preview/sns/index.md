## LinkedIn

🔍 "신화라는 이름의 AI" — Anthropic이 만든 것, 그리고 공개하지 않은 것

Anthropic이 Project Glasswing이라는 이름으로 새로운 AI를 공개했습니다.
이름은 Claude Mythos Preview.
Mythos — 그리스어로 '신화'라는 뜻이죠.

그런데 이 AI, 지금 누구도 사용할 수 없습니다.

🔸 OpenBSD 27년·FFmpeg 16년·FreeBSD 17년된 코드에서 제로데이 발견
🔸 인간 전문가가 몇 주 걸릴 분석을 수분 만에 완료
🔸 GPT-4o 대비 취약점 발견률 4.5배, 오탐률 3.2배 낮음

Anthropic은 역대 가장 강력한 사이버 AI를 만들고도, 공개를 거부했습니다.

이것이 프로메테우스 신화의 역전입니다.

프로메테우스는 올림포스의 불을 훔쳐 인간에게 주었습니다.
Anthropic은 불을 만들었으나 — 주지 않기로 했습니다.
"책임 있는 공개"라는 이름으로.

통제의 역설도 있습니다.
방어를 위해 AI가 필요하다면, 공격도 같은 AI를 원할 것입니다.
Mythos를 막을 수 있는 것도 결국 Mythos뿐이라면,
보안 격차는 오히려 공개 여부와 무관하게 벌어질 수 있습니다.

저희가 주목하는 것은 평가 구조입니다.
DataGreenhouse — 에이전트와 평가 환경이 물리적으로 격리된 합성 데이터 공간.
Anthropic이 Glasswing으로 구현한 것, 그리고 저희가 데이터 품질 검증에 적용하는 원리는 같습니다.

AI에 신화라는 이름을 붙인다면, 어떤 이야기가 펼쳐질까요?

Anthropic이 Project Glasswing을 통해 Claude Mythos Preview를 발표했습니다.

문제는 이 AI가 지금 공개되지 않는다는 것입니다.

Anthropic은 이유를 이렇게 설명합니다.
"공격적 사용 가능성이 방어적 이점을 초과한다."
