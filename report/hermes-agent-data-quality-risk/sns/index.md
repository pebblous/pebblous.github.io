Nature 논문은 합성 데이터가 전체의 1/1000에 불과해도 모델 붕괴가 유발됨을 실증했다. 기존 오염 탐지기는 무작위 추측 수준에 머문다. 2026년 8월 EU AI Act 전면 시행을 앞두고, 자율 학습 데이터의 문서화 의무가 현실화된다.

## LinkedIn (EN)

hermes-agent acquired 58,400+ GitHub stars in just 6 weeks. It is the only agent framework with a triple autonomous learning loop: auto-skill generation, user profiling, and persistent memory. But this self-improvement architecture internalizes three structural risks that silently corrupt training data.

First, Feedback Loop Contamination -- model outputs become training inputs, accumulating bias. Second, Distribution Shift -- overfitting to specific users collapses generalization. Third, Error Fossilization -- early errors become permanent skills, making self-correction impossible.

A Nature paper demonstrated that synthetic data comprising just 1/1000th of total data can trigger model collapse. Existing contamination detectors perform no better than random guessing. With the EU AI Act taking full effect in August 2026, documentation obligations for autonomous learning data are becoming reality.

Pebblous provides infrastructure for pre-diagnosing and continuously monitoring data quality in autonomous learning agents through DataClinic. 60,000 stars are concealing this risk.
