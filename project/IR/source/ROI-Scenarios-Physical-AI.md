# Physical AI 데이터 그린하우스: 산업별 ROI 시나리오

**작성일: 2025-11-12**

본 문서는 페블러스의 '데이터 그린하우스' 솔루션이 모빌리티, 조선, 국방 산업에 적용되었을 때의 구체적인 효과와 기대 ROI를 시나리오 기반으로 분석합니다.

## 시나리오 #1: 모빌리티 - 현대자동차 스마트팩토리 용접 불량 검출

### 🎯 도메인 & 타겟
- **산업**: 모빌리티 (Automotive Manufacturing)
- **고객**: 현대자동차 광명공장
- **적용 영역**: 로봇 용접 품질 관리 및 불량 검출

### 💡 핵심 문제: Problem #1 - 데이터 기근 (Data Scarcity)

#### 현재 상황
현대자동차 스마트팩토리에서 로봇 용접 공정은 하루 수천 건의 용접 작업을 수행하지만, **희귀 불량 패턴**은 연간 발생률이 0.001% 미만입니다.

- **데이터 부족**: 실제 불량 사례가 극히 드물어 AI 모델 학습용 데이터 확보 불가
- **결함 검출 한계**: 기존 비전 시스템은 50%의 높은 오탐율(False Positive)로 인해 실질적 활용도 저하
- **비용 손실**:
  - 현재 불량률: 약 16 PPM (Parts Per Million)
  - 자동차 산업 평균 불량 1건당 처리 비용: $500-$1,500
  - 연간 총 생산량이 100만 대라고 가정 시, 차량당 약 1,000개 용접 포인트 존재
  - **연간 불량 처리 비용: 약 $8M-$24M (약 100억-300억원)**

#### 페블러스가 없을 경우의 문제
- 실제 불량 데이터 수집을 위해 수년간 대기해야 함
- 수동 검사 인력 지속 필요 (연간 300+ 시간/애플리케이션)
- 품질 문제로 인한 리콜 및 브랜드 이미지 손실 위험

### 🌱 페블러스 솔루션: Hyper-Synthetic Data 생성

#### 솔루션 적용 방법

**1단계: 현장 데이터 수집 (DataLens @ Edge)**
- 광명공장 용접 로봇 현장에 경량 DataLens 엣지 장비 설치
- 실시간 용접 비전 데이터, 센서 데이터(전류, 온도), 로그 수집
- 정상 용접 데이터 대량 확보 (수백만 건)

**2단계: 합성 불량 데이터 생성 (Hyper-Synthetic @ Core)**
- 고객사 On-Premise 서버에 '데이터 그린하우스' 코어 시스템 설치
- GenAI + CG 기술로 다양한 희귀 불량 패턴 생성:
  - 미세 균열 (Micro-crack)
  - 용접 기공 (Porosity)
  - 언더컷 (Undercut)
  - 스패터 결함 (Spatter defects)
  - 극한 환경 조건 (극저온, 고온, 진동) 시뮬레이션
- 생성 데이터: 정상 데이터 대비 불량 데이터 비율을 1:1로 균형 조정
- **단 2주 만에 100만 건 이상의 고품질 라벨링 완료된 합성 불량 데이터 생성**

**3단계: AI 모델 재학습 & 배포**
- Hyper-Synthetic 데이터로 불량 검출 AI 모델 학습
- On-Premise 환경에서 보안 유지하며 모델 고도화
- 엣지 디바이스에 경량 모델 배포로 실시간 검출 구현

### 📊 기대 효과 & ROI

#### 정량적 효과

| 지표 | 솔루션 적용 전 | 솔루션 적용 후 | 개선율 |
|------|---------------|---------------|--------|
| **불량 검출 정확도** | 50% (오탐 多) | **97-99%** | **+94% 향상** |
| **불량률 (PPM)** | 16 PPM | **3.4 PPM (Six Sigma Level)** | **-78.75%** |
| **데이터 수집 기간** | 3-5년 (실제 불량 대기) | **2주 (합성 생성)** | **99% 단축** |
| **수동 검사 시간** | 월 300시간/앱 | **월 50시간 이하** | **-83%** |

#### ROI 계산

**투자 비용 (Initial Investment)**
- 데이터 그린하우스 On-Prem 라이선스: $500K (약 6.5억원)
- DataLens 엣지 장비 (10개 라인): $200K (약 2.6억원)
- 초기 구축 및 통합 비용: $300K (약 3.9억원)
- **총 투자: $1M (약 13억원)**

**연간 절감 효과 (Annual Savings)**

1. **불량 처리 비용 절감**
   - 기존 불량 처리 비용: 100-300억원
   - 솔루션 적용 후 (78.75% 감소): 21-64억원
   - **연간 절감: 79-236억원**

2. **검사 인력 비용 절감**
   - 월 300시간 → 50시간 (250시간 절감)
   - 검사 인력 시급: $50/hr
   - 10개 라인 적용 시: 250hr × $50 × 10 lines × 12개월 = **$1.5M/year (약 19.5억원)**

3. **리콜 및 워런티 비용 감소**
   - 자동차 리콜 1건당 평균 비용: $500-$1,000/vehicle
   - 불량 78.75% 감소로 인한 리콜 위험 최소화
   - 연간 추정 절감: **$2-5M (약 26-65억원)**

**총 연간 절감액: $82.5M - $257M (약 1,072억 - 3,341억원)**

**ROI 계산**
```
ROI = (연간 절감액 - 초기 투자) / 초기 투자 × 100
    = ($82.5M - $1M) / $1M × 100
    = 8,150% (보수적 추정)

회수 기간 (Payback Period) = 초기 투자 / 연간 절감액
                          = $1M / $82.5M
                          = 약 0.15년 (1.8개월)
```

**🎯 결론: 약 2개월 만에 투자 회수, 연간 8,150%+ ROI 달성**

## 시나리오 #2: 조선 - 한국 조선소 건조 일정 예측 & 최적화

### 🎯 도메인 & 타겟
- **산업**: 조선 (Shipbuilding)
- **고객**: 한국 주요 조선소 (현대중공업, 삼성중공업, 한화오션 등)
- **적용 영역**: 선박 건조 공정 관리 및 일정 예측

### 💡 핵심 문제: Problem #3 - 멀티모달 데이터 복잡성

#### 현재 상황
조선소의 선박 건조는 수천 개의 공정이 복잡하게 얽힌 초대형 프로젝트입니다.

- **데이터 복잡성**:
  - 3D 설계 도면(CAD), 드론 촬영 영상, 용접/도장 센서 데이터, 자재 재고 데이터, 작업자 투입 스케줄 등 **이질적인 멀티모달 데이터**가 실시간으로 발생
  - 각 데이터의 형식, 샘플링 주기, 좌표계가 모두 상이

- **소통 장벽**:
  - 현장 관리자(Non-IT)가 복잡한 데이터를 이해하고 의사결정에 활용하기 어려움
  - AI 개발팀과 현장 간 "해석의 벽" 존재

- **비용 손실**:
  - 한국 조선소는 글로벌 최고 수준이지만, 일정 지연 및 원가 증가 압박 증가 중
  - 미국 조선소의 경우 평균 **57% 예산 초과, 6개월 이상 일정 지연** 사례 다수
  - 대형 선박(LNG선, 컨테이너선) 1척당 건조 원가: $100M-$200M
  - **일정 지연 1개월당 손실: 약 $5-10M (65억-130억원)**

#### 페블러스가 없을 경우의 문제
- 건조 현장의 병목(Bottleneck) 발견이 늦어 일정 지연 누적
- 현장 관리자가 복잡한 데이터를 활용하지 못해 의사결정 지연
- 납기 준수율 하락으로 고객 신뢰도 감소

### 🌱 페블러스 솔루션: PebbloScope 3D 시각화 + Edge-to-Core 아키텍처

#### 솔루션 적용 방법

**1단계: 멀티모달 데이터 수집 (DataLens @ Edge)**
- 조선소 전역에 DataLens 배치:
  - 드론 촬영 영상 수집
  - 용접/도장 로봇의 센서 데이터 수집
  - 크레인 및 운반 장비의 GPS/IMU 데이터 수집
  - 자재 재고 시스템(ERP) 연동
- 실시간 데이터를 경량화하여 Core로 전송

**2단계: 데이터 통합 & 디지털 트윈 구축 (@ Core)**
- On-Premise '데이터 그린하우스' 시스템에서:
  - 이질적인 멀티모달 데이터를 시간 및 공간 좌표 기준으로 동기화
  - 3D CAD 설계 도면을 기반으로 실시간 건조 진행 상황을 디지털 트윈으로 구현
  - Agentic AI가 공정 병목 지점을 자동 탐지 및 예측

**3단계: PebbloScope로 3D 시각화 & 현장 소통 강화**
- **Non-IT 현장 관리자를 위한 직관적 3D 대시보드 제공**:
  - 선박 전체를 3D로 표시하고, 각 블록(Block)별 진행률을 색상으로 구분
  - 센서 데이터, 드론 영상, 작업 스케줄을 3D 공간에 오버레이
  - 병목 구간을 자동으로 하이라이트 및 알림

- **현장-AI팀 간 소통 도구**:
  - 현장 관리자가 3D 뷰어에서 문제 구간을 클릭하면, 관련 데이터와 AI 분석 결과를 즉시 확인
  - "이 블록의 용접 진행이 왜 느린가?"라는 질문에 AI가 자동으로 원인 분석 제공

**4단계: 건조 일정 예측 & 최적화**
- 과거 건조 데이터와 실시간 데이터를 결합하여 **건조율(Construction Rate) 예측 정확도 향상**
- 일정 지연이 예상되는 공정에 자동으로 리소스 재배치 제안

## 시나리오 #3: 국방 - 해병대 전술 훈련 합성 데이터 생성

### 🎯 도메인 & 타겟
- **산업**: 국방 (Defense & Military)
- **고객**: 대한민국 해병대 (기존 레퍼런스 보유)
- **적용 영역**: AI 기반 전술 훈련 시뮬레이션 & 객체 인식 모델 학습

### 💡 핵심 문제: Problem #1 - 데이터 기근 (Data Scarcity) + Problem #2 - 보안 제약

#### 현재 상황
군사 AI 훈련은 극도로 제한된 환경에서 이루어집니다.

- **데이터 희소성**:
  - 실제 전투 상황 데이터는 존재하지 않거나 획득 불가능
  - 특수 환경 (해무, 야간, 극한 기후) 속 적군 객체 인식 데이터 부족
  - 성능 좋은 AI 모델 학습을 위해 최소 5,000만 건 이상의 데이터 필요

- **보안 제약**:
  - 군사 데이터는 **국가 안보 최고 등급 보안**으로 클라우드 반출 절대 불가
  - 모든 데이터 처리는 군 내부망(On-Premise)에서만 가능
  - 외부 데이터 라벨링 서비스 활용 불가

- **비용 손실**:
  - 실제 야외 훈련 비용:
    - F-35 전투기 1시간 비행 훈련: $30,000/hr
    - UH-60 헬기 1시간 훈련: $1,000/hr + 시뮬레이터 절감액 $304/hr
    - 해병대 상륙작전 훈련(실탄, 연료 포함): **수억 원/회**
  - 데이터 수집 및 라벨링 외주 비용: 수십 년 소요 + 수백억 원
  - **시뮬레이션 기반 훈련은 실전 훈련 대비 비용이 1/20 ~ 1/24 수준**

#### 페블러스가 없을 경우의 문제
- 실제 전투 데이터 부재로 AI 모델 성능 저하
- 외부 반출 불가로 데이터 라벨링 병목
- 고비용 실전 훈련 지속 필요

---

### 🌱 페블러스 솔루션: Hyper-Synthetic Data + Agentic Data Clinic (On-Premise)

#### 솔루션 적용 방법

**1단계: 기존 제한된 데이터 수집 & 품질 진단 (Agentic Clinic @ Core)**
- 해병대 내부망에 '데이터 그린하우스' On-Prem 시스템 구축 (보안 승인 완료)
- 기존 보유 데이터 (드론 촬영, 야간 훈련 영상 등) 수집
- **Agentic AI가 자동으로 데이터 품질 진단**:
  - 노이즈 제거, 이상치 탐지, 라벨 오류 수정
  - AI-Ready 형식으로 자동 변환

**2단계: Hyper-Synthetic 전투 시나리오 데이터 생성**
- GenAI + CG 엔진으로 다양한 전술 시나리오 합성 데이터 생성:
  - **극한 환경**: 해무, 폭우, 야간, 사막, 설원
  - **적군 객체**: 차량, 무기, 인원, 위장 상태
  - **다양한 각도 & 거리**: 드론 고도 50m~500m, 수평/수직 각도 변화
  - **Long-tail 시나리오**: 매복, 기습, 위장 공격 등 희귀 전술 상황

- **생성 규모**:
  - 2주 내 **5,000만 건 이상의 고품질 라벨링 완료 합성 데이터** 생성
  - 기존 수십 년 걸릴 작업을 단 2주로 단축

**3단계: AI 모델 학습 & 시뮬레이션 통합**
- Hyper-Synthetic 데이터로 객체 인식, 전술 예측 AI 모델 학습
- On-Premise 환경에서 보안 유지하며 모델 고도화
- 군 시뮬레이션 시스템에 통합하여 **리얼리즘 향상된 가상 훈련 환경** 제공

#### ROI 계산

**투자 비용 (Initial Investment)**
- 데이터 그린하우스 On-Prem 라이선스 (군용 보안 강화): $800K (약 10.4억원)
- Hyper-Synthetic 데이터 생성 엔진: $500K (약 6.5억원)
- Agentic Clinic 구축 및 통합: $700K (약 9.1억원)
- **총 투자: $2M (약 26억원)**

**연간 절감 효과 (Annual Savings)**

1. **실전 훈련 비용 절감**
   - 기존 연간 실전 훈련 비용: $50M (약 650억원) 가정
   - 시뮬레이션으로 대체 가능 비율: 50%
   - 시뮬레이션 비용은 실전의 5-20%
   - **연간 절감: $20-23.75M (약 260-309억원)**

2. **데이터 라벨링 비용 절감**
   - 외부 라벨링 서비스 비용 (가상): $10M/year
   - Hyper-Synthetic 자동 라벨링: $0
   - **연간 절감: $10M (약 130억원)**

3. **AI 모델 성능 향상으로 인한 전술 효과**
   - 객체 인식 정확도 향상 → 전술 의사결정 속도 및 정확성 증가
   - 정량화 어려우나, **작전 성공률 향상 및 인명 피해 감소** 효과
   - 보수적으로 **연간 $5-10M 상당의 운영 효율화** 추정

**총 연간 절감액: $35M - $43.75M (약 455억 - 569억원)**

**ROI 계산**
```
ROI = (연간 절감액 - 초기 투자) / 초기 투자 × 100
    = ($35M - $2M) / $2M × 100
    = 1,650% (보수적 추정)

회수 기간 (Payback Period) = 초기 투자 / 연간 절감액
                          = $2M / $35M
                          = 약 0.69개월 (약 8.3개월)
```

**🎯 결론: 약 8개월 만에 투자 회수, 연간 1,650%+ ROI 달성**
