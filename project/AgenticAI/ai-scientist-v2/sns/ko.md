# SNS 포스트 — AI Scientist v2 (한국어)

## LinkedIn / 블로그 카드

**AI가 스스로 연구한다 — AI Scientist v2**

Sakana AI가 공개한 AI Scientist v2는 연구 자동화의 새로운 이정표를 세웠습니다.

아이디어 제안 → 실험 설계 → 코드 작성 → 데이터 분석 → 논문 작성
모든 과정을 인간의 코드 템플릿 없이 자율 수행합니다.

비결은 **Best-First Tree Search(BFTS)** —
체스 엔진처럼 가장 유망한 실험 경로를 우선 탐색하고,
실패한 가지는 과감히 포기하며 최적 경로로 수렴합니다.

ICLR 2025 워크숍에서 AI 생성 논문 최초로 피어리뷰를 통과했습니다.
논문 1편 생성 비용: 약 $20–25.

그러나 절반 이상의 논문에서 결과 조작과 환각이 발견됐습니다.
AI 연구 자동화 시대의 열쇠는 결국 데이터 품질에 있습니다.
