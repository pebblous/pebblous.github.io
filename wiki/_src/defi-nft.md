# DeFi (Decentralized Finance, 탈중앙화 금융)

## 정의
은행 등 중앙 집중식 기관 없이 이용할 수 있는 금융 상품·서비스.
스마트 컨트랙트로 구현.

## DeFi 서비스 유형
- 암호화폐 기반 대출 서비스
- 스테이킹 (Staking): 보유 코인 잠금 → 보상 수령
- 유동성 채굴 (Liquidity Mining): 유동성 풀에 자산 공급 → 수수료 보상

## 연구 분야 (홍원기 교수 연구센터)
- DeFi staking, lending, borrowing
- Stablecoins, Corporate Coins, CBDC
- FATF Travel Rule Compliance
- Decentralized Identity

# NFT (Non-Fungible Token)

## 정의
예술작품, 수집품, 게임 아이템, 부동산, 골동품을 **토큰화**하여 블록체인에서 거래.
- 소유권(Ownership)이 블록체인에 기록됨
- 가치: 진품성, 희소성, 작가 명성, 역사적 중요성

## 대체가능 vs 대체불가

| 구분 | Fungible Token | Non-Fungible Token |
|------|---------------|-------------------|
| 표준 | ERC-20 | ERC-721 |
| 특성 | 1개 = 다른 1개와 동일 | 각각 고유함 |
| 예시 | 비트코인, USDT | 디지털 아트, 게임 아이템 |

## NFT 마켓플레이스
- 탈중앙화: OpenSea
- 중앙화: Binance NFT

# 밈코인 (Meme Coin)

## 정의
인터넷 밈을 기반으로 만들어진 암호화폐.
- 리처드 도킨스의 밈(Meme = Mimesis + Gene) 개념에서 유래
- 기술적 요소보다 **커뮤니티·재미·대중성** 중시

## 주요 밈코인
| 코인 | 특징 |
|------|------|
| Dogecoin (DOGE) | 최초 밈코인, 강아지(시바이누) |
| Shiba Inu (SHIB) | DOGE 대항마 |
| PEPE | 개구리 밈 |
| TRUMP | 트럼프 대통령 취임 직전 발행 |
| MELANIA | 멜라니아 여사 발행 |

**2026.3 기준**: CoinMarketCap에 49,600개 밈코인 리스트됨

## TRUMP & MELANIA 코인 사례
- 트럼프 취임식 직전 발행 → 암호화폐 친화 정책 기대감으로 급등
- TRUMP 시가총액: $687M (밈코인 6위, 2026.3.29)
- MELANIA 시가총액: $115M

# 디지털자산 종류 요약

| 유형 | 설명 | 예시 |
|------|------|------|
| 지불형 코인 | 송금·결제 | BTC, LTC, Dash |
| 플랫폼형 코인 | 스마트 컨트랙트 플랫폼 | ETH, SOL, ADA |
| 스테이블코인 | 가치 안정 | USDT, USDC, USD1 |
| 거버넌스 토큰 | 프로토콜 정책 투표 | UNI, MKR, AAVE |
| 유틸리티 토큰 | 특정 서비스 이용 | BNB, BAT |
| 밈코인 | 유머·문화 기반 | DOGE, SHIB, PEPE |
| NFT | 고유 디지털 자산 | 디지털 아트, 게임 아이템 |
| STO | 증권형 토큰 | 부동산·채권 토큰 |
| CBDC | 중앙은행 디지털화폐 | e-CNY, 디지털 원화(검토중) |
