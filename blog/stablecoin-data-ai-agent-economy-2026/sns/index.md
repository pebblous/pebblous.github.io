## Facebook

지난해 스테이블코인이 처리한 결제액이 33조 달러를 넘어섰다. Visa와 Mastercard의 연간 합산 처리량을 초과한 수치다.

그런데 이 거래의 상당 부분을 사람이 하지 않았다. AI 에이전트가 9개월 동안 약 1억 4천만 건을 직접 결제했다. API 호출 비용을 내고, 데이터를 구매하고, 컴퓨팅 자원을 빌렸다. 카드 번호를 입력하거나 결제 버튼을 누른 사람은 없었다.

이 흐름은 이미 인터넷의 작동 방식을 바꾸고 있다. HTTP 요청 하나에 결제 정보를 담아 자동으로 처리하는 표준이 등장했고, Cloudflare와 Stripe, AWS가 이미 도입했다. 웹 크롤러가 콘텐츠를 읽을 때마다 건당 결제하는 구조도 시범 운영 중이다.

한국 시장은 어디에 있는가. Visa는 최근 한국을 "스테이블코인 실험의 최적 테스트베드"라 공개 언급했다. 국내 주요 은행들과 토스는 원화 스테이블코인 발행을 준비하고 있고, 카카오페이는 관련 국제 표준의 창립 멤버로 참여했다. 규제는 아직 완성되지 않았지만, 참여자들은 이미 움직이고 있다.

스테이블코인의 실체와 숫자의 의미, 각국 규제 현황, 그리고 데이터 비즈니스의 수익 모델이 어떻게 달라지는지를 정리했다.

## LinkedIn

지난해 스테이블코인 결제 처리액이 33조 달러를 기록했다. Visa와 Mastercard의 연간 합산을 넘어선 수치다.

숫자보다 더 주목할 것은 누가 결제했느냐다. AI 에이전트가 9개월 동안 약 1억 4천만 건을 자율적으로 처리했다. 사람의 개입 없이, 구독 계정 없이, API 호출 한 번에 결제가 완결됐다.

이 구조가 데이터 비즈니스에 미치는 함의는 구체적이다.

월정액 구독 모델이 쿼리당 결제 모델로 전환된다. 에이전트는 필요한 만큼만 산다. 과금 단위가 '월'에서 '요청 1건'으로 바뀐다.

Cloudflare는 웹 크롤러가 페이지를 읽을 때마다 건당 비용을 부과하는 pay-per-crawl을 시범 운영 중이다. Nansen과 Messari는 블록체인 분석 데이터를 에이전트가 요청 단위로 구매하도록 전환했다. Stripe는 스테이블코인 결제와 기존 카드 결제를 동일 인터페이스로 통합했다.

한국 시장의 변수는 원화 스테이블코인이다. 국내 은행권과 핀테크가 발행을 준비 중이고, 카카오페이는 국제 표준 거버넌스에 창립 멤버로 참여했다. 규제 완성 시점이 이 시장에서의 경쟁 우위를 결정할 가능성이 높다.
