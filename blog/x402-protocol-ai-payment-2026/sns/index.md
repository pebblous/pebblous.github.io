## Facebook

지난 4월 2일, Google과 AWS, Microsoft, Stripe, Visa, Mastercard, 그리고 카카오페이가 하나의 표준에 창립 멤버로 이름을 올렸다. 이름은 x402. HTTP 상태 코드 402번을 부활시켜 AI 에이전트의 자율 결제 표준으로 만든 프로토콜이다.

코드 402는 1991년 인터넷이 설계될 때 "언젠가 유료 인터넷이 올 것"이라는 기대로 예약해둔 자리였다. 신용카드와 구독 모델이 주류가 되면서 30년 이상 사용되지 않았다. Coinbase가 2025년 5월 이 코드를 꺼내들었고, 6개월 만에 1억 건 이상의 결제가 처리됐다.

작동 방식은 단순하다. AI 에이전트가 유료 API에 요청을 보내면 서버가 402로 응답한다. 응답에는 금액, 수신 주소, 체인 정보가 담겨 있다. 에이전트는 자동으로 USDC를 송금하고 데이터를 받는다. 계정 가입도, API 키 관리도, 사람의 클릭도 필요 없다.

미국의 로봇 개발사 OpenMind의 로봇 개는 배터리가 부족해지자 스스로 충전소를 찾아가 플러그를 꽂고 전기요금을 x402로 결제했다. 사람이 시킨 것이 아니다.

Cloudflare는 이 방식으로 웹 크롤러에게 페이지당 요금을 부과하기 시작했다. Vercel은 Next.js API 라우트에 코드 한 줄로 유료 게이트를 달 수 있는 미들웨어를 배포했다. Nansen과 Messari는 블록체인 분석 데이터를 에이전트가 건당 구매하도록 전환했다.

한국에서 카카오페이가 창립 멤버로 참여한 것은 이 표준이 단순한 해외 이야기가 아님을 시사한다. 원화 스테이블코인 발행 논의가 진행 중인 지금, x402가 한국 데이터·API 비즈니스에 어떤 변화를 가져올지 분석했다.

## LinkedIn

HTTP 상태 코드 402번이 30년 만에 실제로 사용되기 시작했다.

4월 2일, x402 프로토콜이 Linux Foundation에 편입됐다. Google, Microsoft, AWS, Stripe, Visa, Mastercard, 카카오페이 등 20개 이상 기업이 창립 멤버로 참여했다. Coinbase가 2025년 5월 공개한 이 표준은 6개월 만에 1억 건 이상의 결제를 처리했다.

원리는 간단하다. 유료 엔드포인트에 요청이 오면 서버가 402로 응답하고, 클라이언트(에이전트)는 자동으로 USDC를 송금한 뒤 재요청한다. 계정도, API 키도, 사람의 개입도 없다.

데이터 비즈니스 관점에서 실무적 함의를 정리하면 세 가지다.

첫째, 구독 모델이 쿼리당 결제로 이동한다. AI 에이전트는 필요한 만큼만 사고, 과금 단위는 '월'에서 '요청 1건'이 된다.

둘째, MCP 서버의 개별 도구에 가격표를 붙이는 것이 가능해진다. 에이전트가 유료 도구를 호출하면 자동으로 결제가 처리된다. 서버 측 구현은 미들웨어 한 줄이다.

셋째, 카카오페이가 창립 멤버로 참여한 만큼 원화 스테이블코인 도입 시 한국 서비스의 통합 경로가 열려 있다.

DataClinic 같은 데이터 품질 진단 서비스가 AI 에이전트 파이프라인에 자동으로 삽입되는 구조, 그 인프라의 표준이 지금 완성되고 있다.
