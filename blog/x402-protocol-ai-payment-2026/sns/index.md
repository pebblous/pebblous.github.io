## LinkedIn

🚀 30년 잠든 HTTP 코드가 AI 에이전트의 지갑이 됐습니다.

1991년 인터넷 설계자들이 "언젠가 쓰일 것"이라며 예약해 둔 HTTP 402 — "Payment Required." 그 코드가 2025년, AI 에이전트 자율결제 표준으로 부활했습니다.

x402의 작동 방식은 단순합니다.
서버가 402로 응답하면, 에이전트가 USDC를 즉시 송금하고 API 접근권을 얻는 것이죠.
계정 가입도, API 키도, 사람의 개입도 없습니다.

이미 누적 1억 건을 처리했습니다. 최근 30일만 $24M, 7,500만 건.

2026년 4월, Linux Foundation 편입과 함께 Google·AWS·Microsoft·Stripe·Visa·Mastercard·KakaoPay 등 20여 개 기업이 창립 멤버로 합류했습니다. 에이전트 결제 인프라가 글로벌 표준으로 굳혀지는 순간입니다.

구독 모델이 쿼리당 결제로 바뀌고, 데이터 파이프라인에 결제가 내장됩니다. 이 변화가 데이터 비즈니스에 무엇을 의미하는지 분석했습니다.
